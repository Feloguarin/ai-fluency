# claude-insight
# Private, local AI coding session analyzer
# No network calls, no telemetry, no external dependencies beyond these:

# Core: no pip dependencies. Everything runs on the Python 3.9+ standard
# library, including the Ollama HTTP calls (via urllib).

# Optional, NOT a pip package — for local AI analysis:
#   Install Ollama from https://ollama.com and run:  ollama pull gemma3:4b
# Without it, Claude Insight falls back to heuristic analysis automatically.
